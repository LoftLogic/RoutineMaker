
Instructions for building the and running the project:
This is a local terminal program that uses JavaScript. As such, all that is needed to run the program is any terminal, and node.js.
To run the program, navigate to the directory of this file and type:
    node terminal.js
You can download Node.js here: https://nodejs.org/en/download/package-manager
This program does use some external libraries/modules, but these are not necessary to install to run the program.
